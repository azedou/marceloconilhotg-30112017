Endpoints implemented until now
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
/allStudents
Get all the Students
--------------------------------
/addStudent
Add a new Student
You should pass the student`s RA and NAME
Exemple: /addStudent?ra=123456789?name=JOAO DA SILVA
--------------------------------
/getStudent
Get a specific Student
You should pass the student`s RA
Exemple: /getStudent?ra=123456789
--------------------------------
